CloudChat

CloudChat is an open-source implementation of a web-presented chat application that allows you to stay in touch even when you are not around.

Under the hood

The implementation is bad and I should feel bad. Oh wait, there isn't even one yet! The platform is based on modern technologies including JavaScript, WebSockets, HTML5, CSS3, Python and Flask.

Stay in touch

Connect to your favorite IRC servers and channels and stay there while you are not near your computer. You can then connect from anywhere and resume chatting!

Get the source

This implementation is fully open-sourced, meaning you can go get the source code and host your own cloud bouncer that is full of security holes!
